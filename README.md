## Git Flow
- main : 최종적으로 배포되는 브랜치
- dev : 배포 전, 개발 중심으로 검증 위주의 브랜치
- feature : 추가 기능을 개발 혹은 기존 기능 수정 브랜치
- hotfix : main 브랜치에서 발생한 버그를 수정하는 브랜치
